# Security Policy

## Supported Versions

| Version  | Supported          |
| -------- | ------------------ |
| 0.24.x   | :white_check_mark: |
| < 0.24.x | :x:                |

## Reporting a Vulnerability

Write an email to yapion.yoyosource@nidido.de explaining your vulnerability. In the subject use the word "Vulnerability" for quick reference.
